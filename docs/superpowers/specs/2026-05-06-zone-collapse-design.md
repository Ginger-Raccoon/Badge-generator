# Дизайн: сворачивание зон

## Цель

Добавить возможность сворачивать/разворачивать зоны в боковой панели редактора. По умолчанию зоны открыты. Состояние — только на время сессии, не сохраняется в проекте.

## Состояние

В `ZoneList.jsx` добавляется один хук:

```js
const [collapsedZones, setCollapsedZones] = useState(new Set())
```

Toggle-функция создаёт новый `Set` (иммутабельно) и добавляет или удаляет `zone.id`.

## Шапка зоны

Шапка (`Box` с именем и кнопкой удаления) становится кликабельной и перестраивается:

- Слева: `ExpandMoreIcon` (развёрнуто) / `ExpandLessIcon` (свёрнуто) — визуальный индикатор
- По центру: имя зоны (`ListItemText`)
- Справа: `DeleteIcon` со `stopPropagation`, чтобы удаление не триггерило collapse

Клик по шапке вызывает toggle. `cursor: 'pointer'` на шапке.

## Контент зоны

Все `FormControl`-ы зоны оборачиваются в `<Collapse in={!isCollapsed}>`. При сворачивании MUI даёт плавный slide-up, при разворачивании — slide-down.

## Затронутые файлы

- `src/renderer/components/ZoneList.jsx` — единственный изменяемый файл
  - Новые импорты: `Collapse` из `@mui/material`, `ExpandMoreIcon`/`ExpandLessIcon` из `@mui/icons-material`
  - Новый хук: `useState` для `collapsedZones`
  - Перестройка шапки зоны
  - Обёртка контента в `<Collapse>`

## Ограничения

- Состояние не персистируется — сбрасывается при перезагрузке приложения
- Выделение зоны (`onSelectZone`) и удаление работают независимо от состояния collapse
