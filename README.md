# Бейджик

Приложение для массовой генерации PDF-бейджей из Excel-таблиц и PSD-шаблонов.

## Установка

Скачай актуальную версию на вкладке [Actions](../../actions) → последний запуск **Build** → раздел **Artifacts**.

### Windows

1. Скачай `build-windows-latest.zip`, распакуй
2. Запусти `Setup.exe`
3. При появлении предупреждения SmartScreen — нажми **Подробнее** → **Выполнить**
4. Приложение установится и запустится автоматически

### macOS

> **Ограничение:** приложение не подписано сертификатом Apple Developer, поэтому macOS Ventura и новее блокирует его запуск с сообщением «приложение повреждено». Это ложное срабатывание защиты — приложение безопасно.

1. Скачай `build-macos-latest.zip`, распакуй — получишь `Бейджик-*.dmg` и скрипт `unquarantine-mac.command`
2. **До открытия DMG** дважды кликни `unquarantine-mac.command` (подробности в `README-mac.md` рядом)
3. Открой DMG, перетащи `Бейджик.app` в папку **Программы**
4. Запусти приложение

## Сборка из исходников

Требования: Node.js 22, npm 11

```bash
npm install
npm start        # режим разработки
npm run make     # собрать установочный файл
npm test         # запустить тесты
```
