# Установка на macOS

Приложение подписано ad-hoc подписью (без Apple Developer ID). Если после
скачивания этого артефакта macOS пишет **«Приложение повреждено, переместите
в корзину»** — дело не в повреждении файла, а в том, что Gatekeeper не
доверяет ad-hoc подписи для файлов, помеченных карантином (это происходит
автоматически при скачивании, AirDrop, переносе через флешку/облако и т.п.).

## Как исправить

1. Распакуйте артефакт (если он в zip).
2. Дважды кликните `unquarantine-mac.command` рядом с этим файлом.
   Если Finder не даёт запустить — в Terminal: `chmod +x unquarantine-mac.command && ./unquarantine-mac.command`.
3. Откройте `.dmg` и установите приложение как обычно.

Скрипт снимает атрибут `com.apple.quarantine` со всех `.dmg`/`.app` рядом
с собой — после этого macOS запускает приложение без предупреждений.
